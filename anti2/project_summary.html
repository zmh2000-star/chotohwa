1. 프로젝트 개요

본 프로젝트는 퍼플스 그룹의 2026년 B2B 확장 전략에 따라, 기업 파트너십 유치와 임직원 복지 솔루션 제안을 **디지털화하고 자동화**하는 것을 목표로 합니다. 1단계 시스템 구축을 통해 견고한 인프라를 마련하였으며, 2단계 고도화를 통해 프리미엄 브랜드 가치를 극대화했습니다.

상세 기술 스택 (Target Architecture)

프로토타입 단계를 넘어 대규모 리드 수집 및 분석이 가능한 엔터프라이즈급 스택을 채택하고 있습니다.

구분 기술 스택 비고
Frontend Next.js / Tailwind CSS SSR(서버 사이드 렌더링) 및 SEO 최적화
Backend Node.js (Turbo/Express) 고성능 비즈니스 로직 및 API 서버
Database PostgreSQL (Supabase / Docker) 컨테이너 환경 기반의 데이터 관리
Utilities Supabase SDK / Marked.js 인증, 스토리지 및 AI 분석 보조
Analytics Google Tag Manager (GTM) 사용자 행동 추적 및 마케팅 데이터 수집

2. 시스템 통합 및 데이터 연동

운영 환경 (Infrastructure)

확장성과 유지보수 편의성을 위해 **Docker 컨테이너** 기반의 PostgreSQL 환경을 구축하였으며, Supabase 클라우드 인프라를 통해 전 세게 어디서나 안정적인 데이터 접근이 가능합니다.

DB 테이블 상세 설계 (leads)

수집되는 모든 리드는 다음과 같은 정규화된 스키마를 통해 안전하게 관리됩니다.

컬럼명 (Column) 타입 (Type) 제약/설명 (Description)
id uuid Primary Key (고유 식별값)
created_at timestamp 상담 신청 일시 (자동 생성)
company_name text 기업명 (필수)
name text 담당자 성함 (필수)
phone text 연락처 (필수)
brand text 선택 브랜드 (Purples, D.Noble, Unnie 등)
hr_concern text 주요 인사 고민 (AI 진단 기반)
memo text 영업팀 개별 상담 메모 및 특이사항
utm_source text 유입 채널 (Meta, Google, Naver 등)
status text 진행 상태 (New, In-Progress, Completed)

데이터 필드 구성 및 마케팅 분석

정밀한 인사이트 도출을 위해 다음과 같은 데이터를 세밀하게 수집하고 관리합니다.

기본 정보 필드

기업명, 업종, 담당자 직함, 연락처 등 핵심 영업 리드 정보 자동 매핑.

AI 진단 로그

임직원 연령대, 인사 고민 사항(이직률, 사기 등), 선호 복지 솔루션 데이터.

상담 성격 분석

상담 선호 시간, 제휴 규모, 주요 관심 브랜드(5-Track) 자동 필터링.

마케팅 어트리뷰션

유입 소스(UTM 파라미터), 레퍼러, 디바이스 정보를 통한 광고 효율 추적.

3. 디자인 시스템 혁신 (Premium Persona)

1단계 인프라 완성을 바탕으로, 2단계에서는 퍼플스 그룹만의 브랜드 가치를 시각화하는 최첨단 UI/UX를 구현했습니다.

핵심 시각 인터페이스 및 인터랙션

Airy Mesh Gradient

정적 배경에서 탈피하여 실시간 연산되는 메시 그래디언트를 적용, 현대적인 프리미엄 감성을 구현했습니다.

Glassmorphism 2.0

고해상도 백드롭 블러 처리를 통해 깊이감 있는 카드 레이아웃과 높은 가독성을 동시에 확보했습니다.

Success Story Marquee

3D 미니멀 아바타와 마티(Marquee) 엔진을 결합하여 파트너사의 성공 사례를 한눈에 볼 수 있도록 자동화했습니다.

Responsive Narrative

스크롤 리빌 애니메이션을 통해 콘텐츠가 자연스럽게 서사적으로 등장하여 정보 몰입감을 증대시켰습니다.

영업 기반 마케팅 엔진 (Conversion Optimization)

  • 지능형 5-Track 탭: 각 브랜드별(퍼플스, 디노블 등) 타겟 고객군에 맞춤화된 다중 레이아웃 탭 시스템 구현.
  • 블랙 멤버십 에셋 통합: VVIP 파티 및 컨시어지 영상 에셋을 시스템 내부에 네이티브하게 통합하여 신뢰도 극대화.
