## 🚨 주의사항 (Guard Rails)

### 절대 하지 말 것
- `jangimg/` 폴더의 영상/이미지를 삭제하지 말 것 (HTML에서 직접 참조)
- CDN 라이브러리 버전을 함부로 올리지 말 것 (GSAP/Three.js 호환성)
- `projects.js`의 데이터 구조를 변경할 때 모달 렌더링 로직도 함께 수정

### 반드시 확인할 것
- **새 이미지/영상 추가 시 → `git add` 확인 필수** (Untracked 상태로 빠지기 쉬움)
- 한글 파일명은 Git에서 이슈 발생 가능 → 영문 파일명 권장
- 모바일 반응형 확인 (768px, 480px 브레이크포인트)
